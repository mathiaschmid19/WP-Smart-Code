== Description ==

**WP Smart Code** is a modern, secure WordPress plugin that lets you add custom code snippets to your website without the hassle of editing theme files or risking code conflicts during theme updates.

= Why Choose WP Smart Code? =

* **Safe & Secure** - Add code without breaking your site
* **Theme Independent** - Your snippets survive theme changes and updates  
* **Multiple Languages** - Support for PHP, JavaScript, CSS, and HTML
* **Smart Execution** - Run snippets automatically or use shortcodes
* **Modern Interface** - Clean, intuitive admin panel with syntax highlighting
* **Import/Export** - Backup and migrate your snippets easily

= Getting Started =

1. Install and activate WP Smart Code
2. Go to **Smart Code** in your WordPress admin
3. Click **Add New** to create your first snippet
4. Choose your code type (PHP, JS, CSS, HTML)
5. Write or paste your code
6. Save and activate!

Your custom code is now safely stored in the database and will persist through theme changes and WordPress updates.

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Go to **Plugins > Add New**
3. Search for "WP Smart Code"
4. Click **Install Now** and then **Activate**

= Manual Installation =

1. Download the plugin zip file
2. Go to **Plugins > Add New > Upload Plugin**
3. Choose the zip file and click **Install Now**
4. Activate the plugin

= After Installation =

1. Navigate to **Smart Code** in your WordPress admin menu
2. Click **Add New** to create your first code snippet
3. Choose your code type and add your custom code
4. Save and activate your snippet

== Frequently Asked Questions ==

= Will my snippets be lost if I change themes? =

No! Unlike adding code to your theme's functions.php file, WP Smart Code stores all snippets in the WordPress database. Your code will remain safe through theme changes and updates.

= Can I break my site with a bad code snippet? =

WP Smart Code includes safety features to minimize this risk, but as with any custom code, you should always test snippets on a staging site first. You can also disable snippets quickly if needed.

= What types of code can I add? =

You can add PHP functions, JavaScript code, CSS styles, and HTML content. Each type has its own execution context and syntax highlighting.

= Can I export my snippets to another site? =

Yes! WP Smart Code includes import/export functionality. You can export individual snippets or bulk export multiple snippets as JSON files.

= Is this plugin suitable for beginners? =

Absolutely! The interface is designed to be user-friendly, and you can start with simple snippets like adding Google Analytics code. The syntax highlighting and organization features help you learn as you go.

= How is this different from other code snippet plugins? =

WP Smart Code focuses on modern development practices with a clean interface, REST API support, and professional-grade features while remaining accessible to beginners.
