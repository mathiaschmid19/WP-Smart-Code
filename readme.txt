Safely manage and execute PHP, JavaScript, CSS, and HTML code snippets in WordPress.

== Description ==

WP Smart Code is a powerful, secure WordPress plugin for managing and executing custom code snippets. Similar to WPCode, it provides:

* Safe PHP, JavaScript, CSS, and HTML snippet management
* Conditional execution (by page, post type, user role, etc.)
* Built-in code editor with syntax highlighting
* REST API for programmatic access
* Import/Export functionality
* Gutenberg block integration
* Modern admin interface with React

== Installation ==

1. Upload the plugin folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the WordPress Plugins menu
3. Navigate to WP Smart Code in the admin menu to manage your snippets

== Features ==

* **Secure Execution**: Built-in safety checks and validation before code execution
* **Flexible Conditions**: Execute snippets based on page, post type, user role, and more
* **REST API**: Full REST API support for third-party integrations
* **Modern UI**: React-powered admin interface with CodeMirror editor
* **Import/Export**: Backup and restore your snippets with JSON
* **Gutenberg Compatible**: Insert snippets directly into posts via blocks
